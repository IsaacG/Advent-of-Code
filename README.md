My solutions to a number of coding puzzles.
These are mostly in Python with a handful of Golang.

| Puzzle | Details | Code |
| ------ | ------- | ---- |
| [Advent of Code][aoc]              | The original puzzle. Hard. | [Python][c_aoc_py], [Go][c_aoc_go] |
| [Coding Quest][codingquest]        | Simpler problems. Easy.| [Code][c_cq]    |
| [FlipFlop Codes][flipflop]         | Easy to medium, fun puzzles. | [Code][c_flip]  |
| [Codyssi][codyssi]                 | Good range. Medium. | [Code][c_cody]  |
| [Everybody Codes][everybody.codes] | Close to AoC. Hard. | [Code][c_ec]    |
| [i18n Puzzles][i18n]               | Puzzles centering on internationalization issues. | [Code][c_i18n]  |
| [Marches & Gnats][mng]             | Puzzles that need to be solved by writing a state machine. | [Code][c_mng]   |
| [Tom's Data Onion][data-onion]     | A message wrapped in layers of puzzles. Unwrap layers to find the next layer. | [Code][c_onion] |
| [aquaq][aquaq]                     | | [Code][c_aquaq] |
| [Pi Coding Quest][pi-coding]       | | [Code][c_pi]    |

The `lib/` contains a number of helpers and base runner code.
